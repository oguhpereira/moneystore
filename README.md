## How to run

* Clone repository and install dependencies:

```
$ git clone git@github.com:RocketChat/Rocket.Chat.ReactNative.git
$ cd Rocket.Chat.ReactNative
$ yarn
```

* Run application

```
yarn start
```

## How run test

```
yarn test
```

## How to build

```
yarn build
```
